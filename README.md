# X COSMOS
X Cosmos is a Bubble Shooter game within the given time period.

##*ABOUT*
1. The X Cosmos game consists of multiple levels of increasing difficulty. 
2. The player gets 25 seconds for the first three levels to shoot all the bubbles.
3. For the subsequent levels the player gets 40 seconds.
4. If the player crosses the time bound, he is allowed to complete the level, but cannot move to the next level. The game stops at that point.
5. To play the game again, the player has to re-run the program.

##*INSTRUCTIONS*
1. Shoot the Bubbles with the help of bullets.
2. The player could move by pressiong the keyboard keys.
3. The bullets will be fired on clicking the upper headed arrow key.
4. For first 3 levels you will get 25 sec, otherwise you will not reach the next level.
5. For the next 4 levels you will get 40 sec, otherwise you will not reach the next level.
